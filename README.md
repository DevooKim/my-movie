# 🏢 영화예매감시국

영화예매감시국은 특별관의 새로운 예매 일정이 열리면 Discord로 알려주는 서비스입니다.

영화를 미리 등록하거나 시간표를 계속 확인할 필요 없이, 원하는 특별관의 알림 채널을 켜두고 새 예매 회차가 열릴 때까지 기다리면 됩니다.

## 이용 방법

1. `안내` 채널에서 `🔔 알림 채널 보기` 버튼을 누릅니다.
2. 표시된 영화관별 채널에서 원하는 특별관의 알림을 확인합니다.
3. 새로운 예매 일정이 열리면 해당 채널로 알림이 전송됩니다.
4. 알림의 앱 열기 버튼을 눌러 영화관 앱에서 예매 가능 여부를 확인합니다.

알림 채널은 읽기 전용입니다. 메시지를 작성할 수는 없지만 이모지 반응은 남길 수 있습니다.

## 알림 내용

새 예매 회차가 발견되면 다음 정보를 안내합니다.

- 영화 제목
- 상영 날짜와 시간
- 잔여 좌석과 전체 좌석 수(영화관에서 제공하는 경우)
- 영화관 앱 열기 버튼

영화관 조회에 문제가 생기면 `서버-상태` 채널에서 상태를 안내하고, 정상화되면 복구 소식을 전합니다.

예매 가능 여부와 좌석 정보는 알림을 받은 시점 이후 달라질 수 있으므로 영화관 앱에서 최종 확인해 주세요.

## 직접 운영하기

자신의 Discord 서버에서 운영하려면 [Self-hosted 안내](docs/self-hosted.md)를 참고하세요.
