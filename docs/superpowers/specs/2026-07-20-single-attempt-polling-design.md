# 단일 시도 폴링 설계

## 목표

1분 주기 폴링에서 한 번의 외부 조회 실패가 같은 주기 안의 추가 요청으로 이어지지 않게 한다.

## 결정

공용 HTTP 클라이언트의 기본 최대 시도를 `1`로 둔다. 필요한 별도 도구만 `MaxAttempts`를 명시해 재시도를 선택할 수 있다. 따라서 메가박스 요청이 연결 종료나 5xx로 실패해도 해당 폴링 주기에는 한 번만 요청하고, 다음 1분 주기에 다시 조회한다. CGV는 브라우저 기반 조회이므로 이 HTTP 재시도 설정의 영향을 받지 않는다.

## 검증

기본 HTTP 클라이언트가 5xx 응답에 대해 한 번만 요청하고 오류를 반환하는 회귀 테스트를 둔다.
