# 제공자별 조회 백오프 설계

## 목표

메가박스의 요청 제한 가능성이 있는 조회 오류에만 6분 대기를 적용하고, CGV 브라우저 세션 오류는 다음 1분 버스트에서 바로 복구한다.

## 동작

스케줄러는 제공자 조회 실패를 제공자 ID가 포함된 오류로 감싼다. 버스트 결과에 메가박스 조회 실패가 하나라도 있으면 다음 버스트 기준을 실패 완료 시점 6분 뒤로 둔다. CGV 조회 실패, CGV 준비 실패, 데이터베이스 기록 실패, Discord 전송 실패 및 종료 취소는 정상 1분 경계를 사용한다. CGV 조회 실패 시 제공자는 기존대로 손상된 CDP 세션을 닫고 다음 버스트 준비에서 새 세션을 연다.

## 검증

메가박스 조회 실패만 6분 뒤 경계를 선택하고 CGV 조회 실패와 일반 오류는 다음 1분 경계를 선택하는 테스트를 둔다.
